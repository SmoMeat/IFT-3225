Cette page web a été réalisé dans le cadre du premier travail pratique du cours des Technologies de l'Internet (IFT-3225) à l'Université de Montréal. Le but du projet est de créer une application côté client qui permet de répondre à des questions de bio-informatique. Ce projet est réalisé avec Bootstrap et Javascript.
